## Authentication

You authenticate API requests using API keys or OAuth tokens. Include the authorization header in all requests.

<Steps>
  <Step title="Get API Key" icon="key">
    Generate an API key from your account settings.
  </Step>
  <Step title="Include Header" icon="header">
    Add the key to the Authorization header.
  </Step>
  <Step title="Make Requests" icon="send">
    Use the key for all API calls.
  </Step>
</Steps>

## Projects API

<Request tabs="cURL,Python" show-lines="true">
  ```bash
curl -X GET \
  https://api.kanetornberg.com/v1/projects \
  -H 'Authorization: Bearer YOUR_API_KEY'
  ```
  ```python
import requests

headers = {'Authorization': 'Bearer YOUR_API_KEY'}
response = requests.get('https://api.kanetornberg.com/v1/projects', headers=headers)
  ```
</Request>

<Response tabs="Success,Error" show-lines="false">
  ```json
{
  "projects": [
    {
      "id": "123",
      "name": "My Project",
      "description": "Project documentation"
    }
  ]
}
  ```
  ```json
{
  "error": "Unauthorized",
  "message": "Invalid API key"
}
  ```
</Response>

## SDKs and Libraries

You use official SDKs for popular languages. Download from the developer portal.

<Tabs>
  <Tab title="JavaScript" icon="code">
    ```javascript
const client = new KaneTornbergClient({
  apiKey: 'YOUR_API_KEY'
});

const projects = await client.getProjects();
    ```
  </Tab>
  <Tab title="Python" icon="terminal">
    ```python
from kanetornberg import Client

client = Client(api_key='YOUR_API_KEY')
projects = client.get_projects()
    ```
  </Tab>
</Tabs>

<Callout kind="info">SDKs simplify integration and handle authentication automatically.</Callout>
